## Списки

Ненумерованные списки добавляются при помощи выделения пунктов звездочкой. Например вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованный список, необходимо пункты просто пронумеровать.
Например вот так:
1. Первый пункт
2. Второй пункт
